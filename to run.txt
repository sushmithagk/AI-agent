1. Open -> D drive
	-> AI agent folder
	-> web-ui folder

2. Go to address bar
   search -> cmd
	  -> activate venv {command -> .venv\Scripts\activate }
	  -> generate IP address for AI agent

3. Copy the URL and Paste in browser

4. AI agent opened 

5. Go to LLM settings 
   select -> google (LLM Provider)
	  -> API Key 
 
6. Now the AI agent is ready 

Important :
	To get the API key -> go to google studio -> create API key -> copy the API key
